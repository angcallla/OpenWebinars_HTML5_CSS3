Agrupando atributos

Si hay ciertos campos del formulario que pueden ser agrupados utilizo la etiqueta "fieldset".

Esta tiene la siguiente estructura:

  • "fieldset"
    • "legend" para darle un título a la agrupación
    • "controles" donde pongo todo los inputs y etc. No se pone la etiqueta como tal

Atributos interesantes

  • required: campo obligatorio en el form
  • disable: deshabilita el campo
  • readonly: solo lectura, no modificable
  • placeholder: agregar texto introductorio. Ejemplo: Insertar nombre..
  • value: para que tenga un valor en específico
  • tabindex: para que al darle al tabulador, siga un orden concreto
  • multiple: elegir varias opciones en el select
