CSS3: Selectores css

Los selectores son reglas o patrones que me van a permitir seleccionar distintos elementos de mi página web para cambiar sus propiedades o estilos visuales.

Formato de ficheros html

  • selector{
    • prop1: valor1;
    • prop2: valor2;
    • ...
    • propn: valorn;
  • }

Selectores más importantes

  • * (selector universal, selecciona todas las etiqeutas de la página web)
  • #id (solo elementos que tengan como id lo que yo ponga tras el #)
  • .class (solo etiqeutas que tengan de clase lo que yo ponga tras el .)
  • etiqueta (esa etiqueta)
  • selector1, selector2 (Se cambian a los elementos seleccionados por cada selector)
  • selector1 selector2 (Cambia el aspecto de los elementos que cumplan con el selector2 que estén dentro de aquellos que cumplan con el selector1)
  • selector1>selector2 (Cambia el aspecto de los elementos que cumplan con el selector2 y que sean hijos directos del selector1)
  • selector1+selector2 (Cambio los de selector2 que están justo después de los de selector1)
  • selector1~selector2 (Cambio los de selector2 que están justo antes de los de selector1)
  • [atributo expr valor] (Cambio aquellos cuya expr cumpla con el valor)
    • Expr (=,~=,|=,$=,*...)
  • Y cualquier combinación de todos los anteriores

Concepto de Cascada en CSS

Cascada siginifica dos cosas:

  • Los estilos se van propagando hacia abajo. Por ejemplo: si tengo table con fondo verde, todas las celdas o td/th van a estar en verde
  • Si hay una regla que se puede aplicar al mismo elemento y hay conflicto, se aplica lo más específico. Por ejemplo: Si tengo table en verde pero especifico un td/th como amarillo, se aplica el amarillo
