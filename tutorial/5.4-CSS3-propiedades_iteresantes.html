CSS3: Propiedades intresantes

La lista de propiedades css es interminable, así que vamos a ver las más interesantes para empezar

Colores

  • Selector{
    • color: red (nombre en inglés)
    • o
    • color: #FFFFFF (eso es blanco)
    • o
    • color: rgb(255,255,255) (blanco)
  • }

Fondos

  • Selector{
    • background: (me permite modificar todos los aspectos del fondo)
    • background-color: (solo el color)
    • background-image: (para añadir una imagen a mi fondo)
    • background-repeat: (si quiero hacer un mosaico con el fondo, cuantas veces se va a repetir)
    • background-origin: (Defino desde el origen que es la esquina superior izquierda, donde empieza a repetirse el fondo)
  • }

Dimensiones y unidades

  • Selector{
    • widht: (ancho)
    • height: (alto)
  • }

Maneras de representar la altura y anchura

  • px: en píxeles
  • %: en porcentaje de lo que ocupe el padre dentro del árbol
  • em: En relación al tamaño por defecto de la letra del navegador
  • rem: En relación al tamaño por defecto de la letra que tiene la etiqueta HTML

Márgenes, bordes y paddings

Márgenes y paddings

Igual para margin que para padding

  • Selector{
    • // A - D - AB - IZQ
    • margin: 20px 50 px 20px 50 px;
    • // A y AB, y DCHA e IZQDA
    • margin: 20px 50px;
    • // TODOS
    • margin: 50px;
    • o
    • margin-left:
    • margin-top:
    • margin-bottom:
    • margin-right:
  • }

Bordes

  • Selector{
    • border: 1px solid black; (todas las propiedades del borde: grosor, forma y color)
    • border-color: black; (solo el color)
    • border-width: 1px; (solo el grosor)
    • border-style: solid; (solo la forma)

    • Y las mismas propiedades para
    • border-left:
    • border-right:
    • border-top:
    • border-bottom:
  • }

Texto

  • Selector{
    • font-family: (cambiar formato de letra)
    • font-weight: (poner negrita, cursiva, ...)
    • font-size: (tamaño de letra)
    • ....
    • text-align: (alineación del texto)
    • text-decoration: (subrayar, tachar, ...)
    • text-indent: (tabular texto)
    • text-transform: (convertir mayusculas y minusculas)
    • ....
  • }
