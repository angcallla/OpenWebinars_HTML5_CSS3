+Etiquetas

Multimedia

Estructura común tanto para audio/video

  • audio/video
    • source (tipos de archivos, tantos como tipos queramos compartir de un mismo archivo)
    • track (tantas como elementos adicionales queramos añadir: subtitulo, ...)

Atributos de "audio" y "video"

  • controls: para mostrar los botones play, foward, ...
  • autoplay: cuando cargues la pagina se activa solo
  • loop: se repite el audio o video
  • muted: mutea

Atributos exclusivos de "video"

  • width: ancho
  • height: alto
  • poster: poner una imagen mientras se está cargando el vídeo

Atributos de "source"

  • src: donde está el fichero
  • type: su tipo

Atributos de "track"

  • src
  • kind: tipo de archivo que estoy enlazando
  • srclang: donde definimos el lenguaje
  • label: descripción
  • type: tipo del archivo

Semánticas

Sirve para decir al navegador, buscador y desarrollador que tipo de contenido va en una sección

Algunas de estas etiquetas son:

  • header: cabecera de mi página web
  • footer: pie de mi página web
  • main: donde está el grueso de mi información, la info principal
  • section: cada una de las secciones que yo pueda distinguir en main
  • article: zonas de la web que tienen sentido por si mismas (por ejemplo: artículo de periódico)
  • aside: nos da información adicional sobre la zona en la que estamos
  • details: para los detalles de una zona en concreto
  • dialog: para cuadros de diálogo
  • summary: para resumenes
  • nav: si tengo un menú de navegación

Características

  • Novedad de HTML5
  • Se comportan como las etiquetas div, es decir, no aportan contenido sino significado. Esto quiere decir que si pones una etiqueta, por ejemplo header, por si sola no vas a ver nada
  • Se utilizan para maquetar
  • Tiene reglas de anidación, todas distinas. Por ejemplo: No puedo tener más de una etiqueta main en mi página

Otras etiquetas

Exiten infinidad de ellas, pero estas son algunas importantes que debes conocer:

  • iframe: me permite meter una página web dentro de una página web
  • canvas: me permite dibujar un lienzo en página web y dibujar cualquier cosa en él. Necesario javascript
  • div: etiqueta que se usa por antonomasia para maquetar además de las Semánticas del apartado anterior
  • span: es una etiqueta en línea, me sirve para distinguir elementos en una linea de mi navegador
  • script: me permite enlazar ficheros normalmente escritos en javascript
  • object/embed: me permite ver ficheros como pdf y cualquier otro utilizando plugins de terceros
