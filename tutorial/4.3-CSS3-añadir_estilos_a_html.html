CSS3: Añadir estilos a html

Hay varias formas:

Manera inline

Consiste en añadir el atributo "style" en la etiqueta que se quiera cambiar el estilo.

Ejemplo:

Texto en color rojo

Inconvenientes:

Si quisiera que todas las etiquetas h3 tuvieran el mismo estilo, tendría que cambiarlas una por una para que ocurra. Esto no es muy óptimo ni recomendable.

Etiqueta style

Se recomienda utilizar en la cabecera html (header)

De esta manera ahora en esta página, todos los h4 van a ser verdes

Ejemplos:

Texto en verde sin hacer nada

Otro texto en verde

Todos los h4 serán verdes

Inconvenientes:

Si mi página web es un conjunto de paginas, tendría que añadir la etiqueta style en el a todas ellas.

@import

Se trata de añadir en la etiqueta style del header: @import 'css/import.css', es decir, localizar la ruta del archivo css que queremos aplicar.

Inconvenientes:

El problema está en la manera en que los navegadores entienden el @import. Hay algunos que no descargan los elementos de la página web hasta que no se ha descargado la hoja de estilos y esto no es muy recomendable si esta hoja fuera muy extensa con lo que dejaría la web sin contenido por mucho tiempo.

Etiqueta link (RECOMENDABLE)

La que todos los autores recomiendan. Vamos a ver ahora sus atributos:

  • href: indicamos la ruta de la hoja de estilos que queremos importar (Ej: 'css/estilos.css')
  • rel: indicamos que tipo de contenido queremos enlazar (En nuestro caso 'stylesheet')
  • type: decimos que se trata de un texto css ('text/css')
